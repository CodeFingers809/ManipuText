# ManipuText

This is a ReactJS app which can help you to manipulate your text as you want! You just enter all of you text and then click on the respective button. Your text will instantly be updated with the function applied! The app has many tools to manipulate the text. All of these tools can be easily used in just the click of a button respective to the tool.

----

# How to use?

1. Clone the repository in a folder
1. Run `npm i` to install dependencies
1. Run `npm run start` to start the development server
